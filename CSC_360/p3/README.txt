How to run:
1. open folder in terminal
2. type `make`
3. SFS is now compiled
4. type `./diskinfo <.IMA file>` and disk info will return
5. type `./disklist <.IMA file>` and will display contents of the .IMA file
6. type `./diskget <.IMA file> <file in root of .IMA>` and file in .IMA root will be copied to the current directory
7. type `./diskput <.IMA file> <file in root of current directory>` and the file will be copied to root of .IMA file (will not go to subdirectory if specified
